# 我也有話要說

### ——普通人的當眾講話技能

這是我在 2010 年 6、7 月份的時候寫的一個系列。牛人就不用讀這個了，這是寫給“普通人”看的。

後來博客網站壞掉了，就只能在 archive.org 上找到。想想有點可惜，就重新整理了一下放在這里。

因為當時並未想著成書，所以是寫到哪里算哪里，別指望在這里看到一個“完整的體系”——這個系列只是“對某些人有一些幫助”而已。
